# Weather Data Visualizer

**Student:** Ayesh Srivastava
**Course:** Programming for Problem Solving using Python

## Overview
This project analyzes local weather data to find trends in temperature, rainfall, and humidity.

## Files Included
- `weather_analyzer.py`: Main analysis script
- `output/`: Contains all generated plots and reports
- `sample_weather_data.csv`: The raw dataset

## How to Run
1. Install requirements: `pip install pandas numpy matplotlib`
2. Run analysis: `python weather_analyzer.py`
